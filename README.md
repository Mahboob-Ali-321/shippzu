# 🚀 Shippzu

A modern food delivery platform built with Next.js, Node.js, PostgreSQL, and Supabase.

## ✨ Features

- 🍔 Restaurant Listing
- 🛒 Smart Cart
- 📍 Live Order Tracking
- 💳 Secure Payments
- 👤 User Authentication
- 🛍️ Restaurant Dashboard
- 🚚 Delivery Partner App
- ⚙️ Admin Panel

## 🛠️ Tech Stack

- Next.js
- React
- Node.js
- PostgreSQL
- Supabase
- Tailwind CSS
