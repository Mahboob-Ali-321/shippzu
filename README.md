# 🚀 Shippzu

A modern food delivery platform built with Next.js, Node.js, PostgreSQL, and Supabase.

A fast, scalable and modern food delivery platform inspired by Zomato and Swiggy, designed for customers, restaurants, delivery partners and administrators.

## ✨ Features

- 🍔 Restaurant Listing
- 🛒 Smart Cart
- 📍 Live Order Tracking
- 💳 Secure Payments
- 👤 User Authentication
- 🛍️ Restaurant Dashboard
- 🚚 Delivery Partner App
- ⚙️ Admin Panel

## 🛠️ Tech Stack

- Next.js
- React
- Node.js
- PostgreSQL
- Supabase
- Tailwind CSS
